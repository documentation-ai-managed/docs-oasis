## Overview

OASIS streamlines your design and development process by integrating seamlessly with popular tools. Connect design software like Figma and Adobe XD for asset imports. Integrate web frameworks such as React and Vue for efficient code generation. Use collaboration features to gather team and client feedback directly in your projects. Set up webhooks for real-time updates, and leverage industry-specific integrations for fintech and edtech workflows.

<Callout kind="info">
  Enable integrations in your OASIS dashboard at [https://dashboard.example.com/settings/integrations](https://dashboard.example.com/settings/integrations).
</Callout>

  <template>

## Webhook Setups

Receive project updates via webhooks to automate workflows.

<Request tabs="cURL,JavaScript" show-lines="true">
  ```bash
  curl -X POST https://api.example.com/webhooks \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer YOUR_TOKEN" \
    -d '{
      "url": "https://your-webhook-url.com/oasis-updates",
      "events": ["design_updated", "feedback_added"]
    }'
  ```
  ```javascript
  await fetch('https://api.example.com/webhooks', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${YOUR_TOKEN}`
    },
    body: JSON.stringify({
      url: 'https://your-webhook-url.com/oasis-updates',
      events: ['design_updated', 'feedback_added']
    })
  });
  ```
</Request>

<Response tabs="200">
  ```json
  {
    "id": "webhook-789",
    "url": "https://your-webhook-url.com/oasis-updates",
    "events": ["design_updated", "feedback_added"],
    "active": true
  }
  ```
</Response>

<ParamField path="url" param-type="string" required="true">
  Your endpoint to receive payloads.
</ParamField>

<ParamField path="events" param-type="array" required="true">
  Array of event types like `design_updated`.
</ParamField>
