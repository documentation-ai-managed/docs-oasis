## Quick Start

Get up and running with OASIS in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create your account at `https://dashboard.example.com` using your business email.
  </Step>
  <Step title="Define Goals" icon="target">
    Outline your project requirements in the strategy dashboard.
  </Step>
  <Step title="Launch Project" icon="rocket">
    Select services and kick off with Harvey Design's team.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
```javascript
// Initialize OASIS client
import { OASIS } from '@oasis/sdk';

const client = new OASIS({ apiKey: 'YOUR_API_KEY' });
await client.strategy.create({ goals: ['scale users', 'improve UX'] });
```
```python
# Initialize OASIS client
from oasis import Client

client = Client(api_key='YOUR_API_KEY')
client.strategy.create(goals=['scale users', 'improve UX'])
```
</CodeGroup>
