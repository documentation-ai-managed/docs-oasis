<Update label="2024-10-15" description="v1.3.0" tags={["feature", "improvement"]}>

## New Features

- Added advanced design strategy tools with AI-powered wireframing and user journey mapping.
- Introduced collaborative canvas for real-time team editing across platforms.

## Improvements

- Enhanced client portal with customizable dashboards and project timeline views.
- Optimized rendering engine for `<50ms` load times on complex designs.

## Bug Fixes

- Resolved sync issues between Figma integrations and OASIS prototypes.
- Fixed export errors for high-resolution PNG and SVG assets.

## Upgrade Instructions

Follow these steps to update OASIS and access the latest features.

<Steps>
  <Step title="Backup Projects" icon="database">
    Export your active projects to avoid data loss.

    ```
    oasis export --projects all --format zip
    ```
  </Step>
  <Step title="Run Update" icon="download">
    Use the CLI to pull the latest version.
  </Step>
  <Step title="Verify" icon="check-circle">
    Test key features like wireframing in a sandbox project.
  </Step>
